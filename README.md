<p>这是一个有spring、springMVC、beetlsql、shiro搭建的maven项目</p>
<p>各个的版本为：<p>
	<p>spring   --4.3.1</p>
	<p>beetlsql --2.9.5</p>
	<p>shiro    --1.2.3</p>
	<p>maven    --3.5.0</p>
	<p>mysql    --5.5</p>
	
<p>1.运行该模板前首先导入src/main/resources/kkbook.sql</p>
<p>2.修改jdbc.properties中默认数据库的用户名与密码</p>
<p>3.项目本地默认地址为localhost:8080</p>
<p>4.默认登录用户名和密码为 root 123456</p>
